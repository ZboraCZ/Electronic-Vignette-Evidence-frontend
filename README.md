# Elektronické dálniční známky

### Dev environment 
* `Node >= 10.16`
* `npm >= 5.6 `

To verify: 
* `npm -v`
* `node -v`

### Available Scripts

Run project:
#### `npm start`

Update project dependencies

#### `npm update`

Test
#### `npm test`

Production bundled build
#### `npm build`
